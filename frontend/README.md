# FitLedger Mobile

Expo React Native app scaffold for the FitLedger migration.

## Start

Install the mobile app dependencies from the workspace root, then run:

```bash
npm --prefix apps/mobile install
npm --prefix apps/mobile run start
```

## Status

This app currently contains the new mobile shell and shared contract wiring. The remaining web screens still need to be migrated from the legacy Next.js frontend.
